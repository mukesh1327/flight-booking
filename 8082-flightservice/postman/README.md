# Postman Files

Import these files into Postman:

1. `flightservice-apis.postman_collection.json`
2. `flightservice-apis-v2.postman_collection.json`
3. `flightservice-local.postman_environment.json`
4. `flightservice-dev.postman_environment.json`
5. `flightservice-prod.postman_environment.json`

Then select one environment (`local`, `dev`, or `prod`) and run requests.

Default URLs:
- local: `http://localhost:8082`
- dev: `http://localhost:8082`
- prod: `https://flight.skyfly.in`
