# Postman Files

Import these files into Postman:

1. `authservice-apis.postman_collection.json`
2. `authservice-apis-v2.postman_collection.json`
3. `authservice-local.postman_environment.json`
4. `authservice-dev.postman_environment.json`
5. `authservice-prod.postman_environment.json`

Then select one environment (`local`, `dev`, or `prod`) and run requests.

Default URLs:
- local: `http://localhost:8081`
- dev: `http://localhost:8081`
- prod: `https://auth.skyfly.in`
