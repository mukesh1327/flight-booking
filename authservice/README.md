# Hello World Authservice

This project is now a minimal Spring Boot hello-world service.

## Run

```bash
./mvnw spring-boot:run
```

On Windows PowerShell:

```powershell
.\mvnw.cmd spring-boot:run
```

## API

- `GET /`
- `GET /hello`

Sample response:

```json
{
  "message": "Hello World"
}
```
